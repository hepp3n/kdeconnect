# Testing on COSMIC Desktop

For testing COSMIC desktop applet, you can build it with Makefile.

# Build

`make build`

# Installing

`sudo make install`

# Uninstalling

`sudo make uninstall`

Make sure you have [rustup.rs](curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh) installed on your system.
